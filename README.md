# Blinkit-Data-Analysis
## Project Overview

This project involves the development of an interactive Power BI dashboard to analyze Blinkit’s sales performance, customer behavior, and inventory distribution. The dashboard transforms raw data into meaningful insights, enabling better decision-making and highlighting areas for business optimization.

## Key Performance Indicators (KPIs)

The analysis centers around the following core KPIs:

Total Sales: $1.20M in overall revenue generated

Average Sales: $141 per transaction

Number of Items Sold: 8,523 unique items

Average Customer Rating: 3.9 out of 5

## Dashboard Features

Dynamic Filter Panel: Enables filtering by outlet location type, outlet size, and item category

Outlet Establishment Trend: Displays outlet growth trends from 2012 to 2022

Fat Content Analysis: Compares sales performance of low-fat versus regular-fat products

Item Category Distribution: Highlights sales contributions across different product categories

Outlet Size & Location Analysis: Evaluates sales performance by outlet size and location tier

Outlet Type Comparison: Assesses outlet types based on total sales, number of items, average sales, customer ratings, and item visibility

## Key Insights & Findings

The business demonstrates strong overall performance, achieving over $1M in total sales

Low-fat products outperform regular options, reflecting a shift toward health-conscious purchasing

Fruits, vegetables, and snack foods emerge as the highest-selling categories

Medium-sized outlets in Tier 3 locations deliver the highest profitability

Supermarkets generate higher sales volumes, while grocery stores show superior item visibility

## Conclusion

This dashboard provides a comprehensive view of Blinkit’s operational performance and customer preferences. The insights derived can support strategic planning, inventory optimization, and targeted marketing initiatives.
